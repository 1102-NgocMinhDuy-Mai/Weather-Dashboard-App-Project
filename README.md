# Weather-Dashboard-App-Project
A responsive web application that displays real-time weather conditions for any city in the world.

Features:
- Search weather by city name.
- Displays temperature (°F), humidity, and weather description.
- Dynamic weather emoji based on conditions.
- Clean, minimal card-based UI.

Built With: HTML, CSS, JavaScript, OpenWeatherMap API.

Getting Started:
Clone or download this repository.
Get a free API key from OpenWeatherMap.
Open index.js and replace the apiKey value with your own key.
Open index.html with Live Server or any browser.

Usage: Type a city name into the search bar and click Get Weather to see the current conditions.
